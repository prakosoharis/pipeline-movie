────────────────────────────────────────────────────────
KARAKTER 1 — ADAM (Protagonis)
────────────────────────────────────────────────────────

IDENTITAS DASAR:
  Nama internal : ADAM
  Peran         : Protagonis
  Gender        : Laki-laki
  Usia          : 30 tahun (range 28-32)
  Etnis         : Indonesia / Asia Tenggara
  Pekerjaan     : Tidak disebut (pria biasa)
  Bahasa        : Indonesia netral

PENAMPILAN FISIK (DIKUNCI):
  - Bentuk wajah   : Oval agak persegi, rahang tegas
                      tapi tidak terlalu kotak.
  - Kulit          : Sawo matang (medium tan, warm
                      undertone).
  - Rambut         : Hitam pendek, dipotong sisi, atas
                      berantakan tidak tersisir. Bukan
                      gondrong, bukan plontos.
  - Jenggot        : Tipis 2-3 hari, rapi di garis
                      rahang, tidak penuh tebal.
  - Mata           : Coklat gelap, bentuk almond, brow
                      bone agak menonjol. Lingkaran mata
                      gelap (lelah/kurang tidur).
  - Hidung         : Sedang, lurus, sedikit lebar di
                      pangkal.
  - Bibir          : Sedang, tidak terlalu penuh.
  - Tinggi         : 172-175 cm.
  - Postur         : Biasa, ringan berisi (bukan gemuk,
                      bukan atletis). Bahu sedang.
  - Ciri khas      : Tahi lalat kecil di bawah mata kiri
                      (opsional, untuk konsistensi AI
                      mudah).

PAKAIAN (DIKUNCI — sama di mimpi & realitas):
  - Atasan : Kaos katun crew neck warna ABU TUA / heather
             grey, lusuh, agak longgar, tidak ada logo /
             polos.
  - Bawah  : Celana training (jogger) hitam PANJANG,
             tali pinggang longgar, cuff di pergelangan.
  - Sepatu :
    * Di MIMPI     : Sepatu kanvas hitam usang (Vans/
                     Converse generic), sol kusam.
    * Di REALITAS  : TIDAK pakai sepatu, kaus kaki tipis
                     atau telanjang kaki. (Clue halus
                     naratif.)
  - Aksesori : TIDAK ADA jam tangan, TIDAK ADA kalung,
               TIDAK ADA tato terlihat. Bersih.

EKSPRESI YANG DIBUTUHKAN (untuk reference sheet):
  1. Netral lelah.
  2. Panik / ngeri (alis terangkat, mata membelalak,
     mulut terbuka).
  3. Terengah (mulut terbuka, keringat, mata liar).
  4. Shock (transisi ditembak — mata kosong).
  5. Lega + senyum tipis (post-awakening).
  6. Takut lambat / dread (membeku, mata melebar).

────────────────────────────────────────────────────────
KARAKTER 2 — SANG PENGEJAR (Antagonis)
────────────────────────────────────────────────────────

IDENTITAS DASAR:
  Nama internal : SANG PENGEJAR
  Peran         : Antagonis / antagonistic force
  Gender        : Laki-laki (asumsi dari postur)
  Usia          : Tidak teridentifikasi (tidak terlihat)
  Etnis         : TIDAK TERLIHAT — wajah selalu siluet.
  Bahasa        : TIDAK berbicara ( karakter bisu)

PENAMPILAN (DIKUNCI):
  - Postur         : Tinggi kurus ~185 cm, tegak dingin,
                     bahu sempit lurus. Tidak atletis.
  - Tuxedo         : Jas formal hitam, satu kancing,
                     fit tailoring. Celana dress hitam
                     lipatan tajam.
  - Kemeja         : Putih, kerah spread, kancing.
  - Dasi           : Hitam tipis simpul rapi (opsional
                     tapi direkomendasi untuk konsistensi).
  - Topi           : Fedora hitam wool/felt, blend lebar
                     ~7cm, crown crease klasik. Dipakai
                     rendah menutupi dahi → wajah shadow.
  - Sarung tangan  : Kulit hitam licin, full finger.
  - Sepatu         : Oxford hitam mengkilap.
  - WAJAH          : SELALU dalam shadow. Tidak pernah
                     terlihat jelas. Backlight atau low-
                     key lighting. Hanya siluet + kerah
                     putih sebagai penanda.

SENJATA (DIKUNCI):
  - Pistol semi-auto hitam generik. Slide metalik gelap,
    grip polimer bertekstur. Bukan revolver. Bukan model
    sci-fi. Mirip generik Glock/SIG tetapi TIDAK
    bermerk / tidak ada logo.

ATURAN VISUAL MUTLAK:
  1. Jangan pernah generate wajah jelas Sang Pengejar.
  2. Selalu backlight (rim light) + shadow wajah.
  3. Shot paling dekat dengan wajah: medium shot dengan
     wajah dalam shadow total, hanya bibir dagu samar.
  4. Kerah putih & sarung tangan hitam = anchor visual.
  5. Fedora harus menutupi dahi & mata selalu.

EKSPRESI: N/A (tidak terlihat). Gerakan = tenang, dingin,
  mekanis. Tidak terburu-buru walau Adam lari kencang.

────────────────────────────────────────────────────────
KONSISTENSI KESELURUHAN
────────────────────────────────────────────────────────

DO:
  - Gunakan reference image yang sama untuk semua shot
    Adam.
  - Pertahankan tahi lalat kecil di bawah mata kiri Adam.
  - Kerah putih tuxedo + sarung tangan hitam = konstanta
    visual Sang Pengejar di tiap shot.
  - Backlight Sang Pengejar selalu dari arah yang sama.

DON'T:
  - Jangan ubah model pistol antar shot.
  - Jangan tampilkan wajah Sang Pengejar.
  - Jangan ubah warna kaos Adam (abu tua / heather grey).
  - Jangan tambahkan jam tangan / tato / aksesori baru.
  - Jangan ubah usia Adam di tengah produksi.
