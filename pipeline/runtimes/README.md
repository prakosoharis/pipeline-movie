# Model Runtimes

Folder ini dicadangkan untuk dokumentasi dan konfigurasi runtime model pada
fase mendatang.

Runtime berbeda dari adapter:

- adapter mendefinisikan cara pipeline berbicara dengan model;
- runtime mendefinisikan environment eksekusi, device, dependency, dan
  resource policy.

Belum ada runtime baru yang diimplementasikan. Runtime Wan2.2 yang sedang
digunakan tetap berjalan melalui script setup dan runner yang ada.
